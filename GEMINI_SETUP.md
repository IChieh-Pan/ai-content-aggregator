# 🤖 Gemini AI Content Curator Setup

Your app now has an AI-powered content curator using Google Gemini! Instead of relying on RSS feeds, Gemini intelligently finds and curates the best UX+AI articles for you.

## 🔑 Get Your Gemini API Key

1. **Go to Google AI Studio**: https://aistudio.google.com/
2. **Sign in** with your Google account
3. **Click "Get API Key"** in the top right
4. **Create a new API key** or use an existing one
5. **Copy the API key** (starts with `AIza...`)

## 🚀 Add to Vercel

1. **Go to Vercel Dashboard**: https://vercel.com/dashboard
2. **Click your project**: `ai-content-aggregator-7oen`
3. **Settings** → **Environment Variables**
4. **Add New Variable**:
   - **Name**: `GEMINI_API_KEY`
   - **Value**: Your API key (starts with `AIza...`)
   - **Environment**: Select "Production"
5. **Save** and **Redeploy**

## ✨ How It Works

- **AI-Powered**: Gemini searches for the latest UX+AI content
- **Smart Curation**: AI evaluates relevance and quality
- **Always Fresh**: New articles curated on each request
- **No Database Issues**: Works independently of database problems

## 💡 Usage

1. **Visit your app**: https://ai-content-aggregator-7oen.vercel.app
2. **Click "🤖 Get AI-Curated Content"** button
3. **Wait 10-30 seconds** for Gemini to curate fresh content
4. **Browse high-quality UX+AI articles** selected by AI
